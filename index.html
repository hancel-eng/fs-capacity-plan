Connect to Double HQ
Your token is saved locally for 23 hours — you only need to do this once per day.
Step 1 — Get your token
Client ID
Client Secret
Run this command in your terminal (Mac: Terminal · Windows: PowerShell):
curl -X POST "https://api.doublehq.com/oauth/token" -H "Content-Type: application/x-www-form-urlencoded" -d "grant_type=client_credentials&client_id=YOUR_CLIENT_ID&client_secret=YOUR_CLIENT_SECRET"
The response will look like:
{"access_token":"eyJhbGc...","token_type":"Bearer"}
Step 2 — Paste your token
Copy the value of access_token from the terminal output and paste it here:
Tag ID → Hours Mapping
Assign an hour value to each Double tag ID found in your tasks. This is saved locally — only needs to be done once per new tag.
